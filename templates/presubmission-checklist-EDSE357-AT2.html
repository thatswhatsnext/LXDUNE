Professional Judgement & Critical Evaluation (35%)
Recommendation if not checked Replace verdict sentences with reasoned ones. Use a sentence frame: "I would [use / not use / adapt] this resource because it [does / does not] ___. In a Year ___ science classroom, this means students would ___." A professional judgement names the consequence of the decision, not just the decision itself.
Recommendation if not checked Even resources you would use have limitations, and resources you would not use may have value in a different context. Add a balancing sentence to each critique: "While this resource is strong in ___, it is limited by ___, which would need to be addressed by ___."
Recommendation if not checked Ask: Is the science in this resource correct? Is it current? Is the source credible? For digital resources especially, a brief comment on accuracy and authority is expected. Use a sentence frame: "The scientific content of this resource is [accurate / outdated / incomplete] because ___."
Recommendation if not checked Name specific cohort characteristics (language background, prior knowledge, access to equipment, cultural context) and explain how the resource does or does not account for them. Use a sentence frame: "For a cohort that includes ___, this resource [supports / creates a barrier] because ___."
Diversity & Intentional Curation (15%)
Recommendation if not checked Use the checklist in the task instructions to confirm coverage: textbook chapter/section, digital resource, first-hand investigation, second-hand data investigation, external expert, assessment item. At least one resource must be applicable for Life Skills. If any type is missing, add a resource before submitting.
Recommendation if not checked The marker wants to see that your collection is intentional, not just compliant. Add a short rationale to your collection — either in an introduction or woven into individual critiques — that explains how the resources work together for a Year 11–12 cohort in your chosen discipline. Use a sentence frame: "I selected this combination because it gives students exposure to ___, which is important for ___ because ___."
Curriculum Literacy & Assessment Alignment (30%)
Recommendation if not checked Copy outcome codes and content dot points directly from the NSW syllabus PDF for your discipline. Do not paraphrase outcomes — the marker needs to see you can read the syllabus accurately. Then add one sentence explaining the link: "This resource addresses [outcome code] because it requires students to ___." Syllabus extracts do not count toward word count.
Recommendation if not checked Senior science is as much about Working Scientifically skills as content knowledge. For each resource, identify the relevant Working Scientifically strand and explain how the resource develops it. Use a sentence frame: "This resource develops [Working Scientifically strand] by requiring students to ___, at a level of [independence / structure] appropriate for Year ___."
Recommendation if not checked For at least some resources — particularly the assessment item — explain what evidence the resource produces and how a teacher would use it to make an assessment decision. Use a sentence frame: "This resource could be used as [formative / summative] assessment because it produces evidence of ___, which can be evaluated against ___."
Scholarly Grounding & Pedagogical Insight (20%)
Recommendation if not checked Replace floating citations with applied ones. Use a sentence frame: "___ (Author, Year) argues that [pedagogical principle]. This resource [does / does not] reflect that principle because ___." Aim for fewer sources used precisely rather than many listed without purpose.
Recommendation if not checked Name a learning theory (constructivism, inquiry-based learning, cognitive load theory, UDL) and explain how it shapes your evaluation of the resource. Use a sentence frame: "From a constructivist perspective, effective resources should ___. This resource [meets / falls short of] that standard because ___." Theory should strengthen your professional judgement — not appear as a separate section disconnected from your critique.
